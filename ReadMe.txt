INSTRUCTION to Install & Uninstall Service

Go to bin\Debug to every folder and copy that path
i.e. D:\Projects\ASP Assignments\ASP-WindowsServices-WebScrapping\k173613_Q1\k173613_Q1\bin\Debug

Open Command Prompt and cd to above path

TO INSTALL ------
Write Command => k173613_Q1.exe install start
Where k173613_Q1.exe is executable file which we install as service to our local machine

TO UNINSTALL ----
Write Command => k173613_Q1.exe uninstall
(Be on same path) This removes service.
